# Real-Time Security Log Visualizer

This project is a real-time security dashboard built with Python and Streamlit, designed to fulfill the requirements of Project 15 for the Cybersecurity class.

## Features
- Monitors `access.log` locally using a File Watcher (Method 2).
- Automatically parses success and failed logins.
- Detects Brute Force attacks (> 5 failed logins).
- Identifies off-hours logins.

## How to Run Locally

1. **Install Dependencies:** Ensure you have Python installed, then run:
   ```bash
   pip install -r requirements.txt
   ```
2. **Start the Log Simulator:** Open a terminal and run the mock log generator (runs continuously):
   ```bash
   python logger_sim.py
   ```
3. **Start the Dashboard:** Open another terminal in the same folder and launch Streamlit:
   ```bash
   streamlit run app.py
   ```

## GitHub Push and Deployment Automation
This project is fully automated for showcase and deployment via the Command Line Interface (CLI):

### Pushing to GitHub
If you make changes to your codebase and wish to update the repository, simply execute:
```bash
git add .
git commit -m "Update project files"
git push origin HEAD
```

### Auto-Deployment to Streamlit Community Cloud (Free)
Streamlit apps easily deploy straight from your Public GitHub repo without needing a server setup. This generates a public URL you can share with your professor:
1. Navigate to [Streamlit Community Cloud](https://share.streamlit.io/).
2. Log in using your GitHub account.
3. Click the **"New app"** button.
4. Search and select this GitHub repository (e.g. `AIT-SecurityLogVisualizer`).
5. In the "Main file path", type `app.py`.
6. Click **Deploy!**
Your cyber-security dashboard is now live on the internet and ready to demo anywhere.
